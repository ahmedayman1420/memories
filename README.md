# Memories
MERN stack social media app

## App features
  O Add Post
  O Edit Post
  O Delete Post
  O Like Post
  O Comment Post
  O Recommend Posts 
  
## It`s built with the following technologies
### Front-end
  #### O react
  #### O axios
  #### O redux & redux-thunk
  #### O react-bootstrap
  #### O react-router-dom
  #### O jwt-decode
  #### O material-ui-chip-input
  #### O rc-pagination
  #### O react-file-base64
  #### O react-google-login
  #### O @fortawesome/react-fontawesome
  
### Back-end
  #### O node.js with express
  #### O mongodb
  #### O JWT
  #### O joi
  #### O rbac
  #### O body-parse
  #### O cors
